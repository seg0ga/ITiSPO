# Диаграмма Saga

project_code: tickets-s05

## Состояния

| **Состояние** |  **Пояснение**   |
|:-------------:|:----------------:| 
|      NEW      |   заказ создан   |
|     PAID      |  заказ оплачен   |
|     DONE      |  заказ выполнен  |
|   CANCELLED   |  заказ отменен   |

## Шаги
1. Создание заказа (**NEW**), бронь билетов
2. Оплата билетов (**PAID**)
3. Завершение после мероприятия (**DONE**)
4. В случае ошибки на любом из этапов - **СANCELLED**

## Компенации

При ошибке оплаты:
- Статус - **CANCELLED**
- Отмена резерва билетов
- Пытаться отменить, пока не получиться
