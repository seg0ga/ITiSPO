В каждой папке (`weeks/week-XX`) находятся:
- **README.md** — что нужно сделать и как запустить.
- **week-XX.md** — немного теории, чтобы погрузиться в тему.
- **questions.md** — ответы не контрольные вопросы.

Основной язык для выполнения заданий — **Python (FastAPI)**
